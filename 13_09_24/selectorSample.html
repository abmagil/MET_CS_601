CSS Selectors

  • ID selector

    Selects the element which has the id specified.

    <style>#selector { font-size: 1.2em; }</style>
    <div id="selector"></div>
  • Element selector

    Selects all elements of the type specified.

    <style>cite { color: red; }</style>
    <cite>Some red text</cite>
  • Class selector

    Selects all elements which include the specified class in their class attribute.

    <style> .italic { font-style: italic; } </style>
    <div class="italic">Italicized</div>
  • Attribute selector

    Selects all elements which have the specified attribute, regardless of its value.

    <style>a[href] { color:green;}</style>
    Contact me.
  • Adjacent selector

    Selects a specified element which is the first sibling of another specified element.

    <style>span + span {font-size: 1.2em;}</style>

    The first element. The second element will get selected.

  • Direct-descendent selector

    Selects a specified element only if it is first-level child of another specified element

    <style>p > em {font-variant: small-caps;}</style>

    This is normal text but now we're talking fancy!

  • Grouped selectors

    When selectors share the same styling, they can be grouped with a comma (,) to save space and increase clarity.

    <style> h3, p { color:#808080; background-color: #0000FF}</style>

    This heading is self contained and styled.

    Boring normal text here

    Fancy paragraph text is styled the same.

  • Descendent selector

    When two selectors are separated only by a space ( ), then the second selector is matched only when it is an nth-child (including 1st) of the first selector

    <style>#match > span {border:#008000;border-style:dashed;}</style>
    The start of the div if unstyled but the part inside the span matches the selector.
  • Universal selector

    The asterisk (*) character matches all elements. It can be placed in a context, such as within another selector or grouped.

    <style>#univ > * {color:#FFFF00;background-color: #FF0000;}</style>

    A heading

    A paragraph

    Even the footer tag, when badly misused!
  • Pseudo classes

    Some elements have additional characteristics beyond simply their existence. Pseudo classes, demarcated as Selector:pseudo-class, allows additional control.

    <style>#link > a:hover {font-size:1.5em;}</style>
    Hover over me and I expand!
  • Pseudo elements

    A set of pseudo-elements are not true elements but act based on properties of the element as displayed in the browser. Common examples are the :first-line or :first-letter pseudo-elements.

    <style>p.pseudo:first-letter {font-family:"Century Gothic";font-size:2em;}</style>

    Some fancy text.
