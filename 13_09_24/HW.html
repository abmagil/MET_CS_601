Part 1

Answer each of the following questions.

1. When should you use an ID selector over a Class selector?

When you are trying to find something that is unique by design, rather than circumstance.

2. What is meant by inheritance in CSS?

Inheritence means that, in the absence of a rule which applies on an element, it will inherit the rules of its parent element.

3. What are CSS At-rules? Give an example of your favorite one.

At-rules are rules which start with the "@" symbol. My personal favorite is @media, which allows styling to be predicated on the medium used to observe a page.

4. What is meant by parent and child? Give an example

These reference the hierarchy of elements in an HTML document. A child tag is entirely enclosed by its parent tag. In the following snippet, the p-tag is the parent and the span-tag the child:

<p>
<span>
These words are in a child element.
</span>
</p>

5. Describe what CSS Vendor Extension Prefixes are, in regards to CSS properties.

Because various browsers can add whatever kind of extension they like to CSS, they are assigned special naming which indicates the vendor. These are free to be implemented however the vendor chooses.

6. What is meant by sibling element?

When two elements are children of the same parent element, they are sibling elements. They need not be adjacent.

7. What does the "Cascade" mean in Cascading Style Sheets?

The cascade in CSS references the way that styles are accumulated from multiple sources all the way down to the element/text level.

8. How many types of Style Sheets are there? Which one has the most weight? Give a description of each one you mention.

There are three types of style sheet. External sheets are linked into documents using the <link> tag. Embedded stylesheets are written directly in documents using the <style> tag. Finally, inline styles (not technically a stylesheet) are written directly in the tags they modify. Inline styles have the most weight, but are also the hardest to debug.

9. What does the !important rule mean. Give a working example of its use.

The !important modifier adds significant weight to whatever rule it is attached to. For instance, see index_1_9.html

10. How do you calculate Specificity. Hint: Use http://reference.sitepoint.com/css/specificity

Specificity is calculated in a hierarchy of weights: !important > inline > id-selectors > class-selectors > element-selectors. A rule will calculate specificity based on all these factors and the highest specificity (counted from the left of the hierarchy) will win. If two share the same specificity, the latter one wins.
